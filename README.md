# django_blog

## Making Blog using Django

### To run this app follow these steps:


`git clone {link}

pip install virtualenv

virtualenv .

activate virtualenv

pip install requirements.txt

python manage.py makemigrations

python manage.py migrate

python manage.py runserver`
